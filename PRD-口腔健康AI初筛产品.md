## 1. 产品定位

### 1.1 目标用户画像

#### 核心用户群（Primary）
| 维度 | 描述 |
|------|------|
| **年龄** | 25-45岁城市白领、新手父母 |
| **痛点** | 工作忙碌没时间定期口腔检查；对孩子口腔健康焦虑；看牙费用高、恐惧感强 |
| **场景** | 居家自测、儿童日常监测、正畸效果跟踪、术前术后恢复观察 |
| **特征** | 智能手机重度用户、健康意识强、愿意尝试AI工具、对隐私敏感 |

#### 次要用户群（Secondary）
| 用户类型 | 需求场景 | 关键诉求 |
|----------|----------|----------|
| **银发族（55-70岁）** | 行动不便，需要远程监测 | 简单操作、子女代管、异常预警 |
| **正畸用户** | 矫正期牙齿变化追踪 | 进度可视化、与医生远程沟通 |
| **下沉市场用户** | 优质口腔医疗资源匮乏 | 低成本初筛、就医指导 |

3分钟居家自测，AI即时预警，专业医生兜底

【核心价值】

### 1.3 竞品差异化定位

| 竞品类型 | 代表产品 | 他们的局限 | 我们的机会 |

## 2. 核心功能模块

### 2.1 AI口腔健康初筛系统

#### 2.1.1 可识别问题清单

| 类别 | 具体项目 | AI检测难度 | 临床价值 |
|------|----------|------------|----------|
| **龋齿检测** | 浅龋、中龋、深龋 | ⭐⭐⭐⭐ | 高（早期发现可避免根管治疗） |
| **牙周健康** | 牙龈红肿、牙龈出血迹象 | ⭐⭐⭐ | 中 |
| **牙结石识别** | 龈上结石、龈下结石（辅助） | ⭐⭐⭐⭐ | 中（提醒洁牙） |
| **牙齿排列** | 拥挤、稀疏、错位、开颌 | ⭐⭐⭐ | 高（正畸需求评估） |
| **牙齿颜色** | 外源性染色、氟斑牙、四环素牙 | ⭐⭐ | 中（美观评估） |
| **缺牙/修复体** | 缺牙、充填体、义齿 | ⭐⭐ | 中 |
| **智齿状态** | 萌出情况、阻生迹象 | ⭐⭐⭐⭐ | 高（拔牙时机判断） |
| **口腔溃疡** | 阿弗他溃疡、创伤性溃疡 | ⭐⭐ | 低（需结合病史） |
| **口腔黏膜** | 白斑、红斑（风险提示） | ⭐⭐⭐⭐⭐ | 高（需医生复核） |

#### 2.1.2 技术实现方案

```
【AI模型架构】

输入层：用户拍摄的口腔照片（标准位：正面、上牙、下牙、左右侧位）
    ↓
预处理：图像质量检测 → 自动对齐 → 区域分割
    ↓
特征提取：CNN骨干网络（ResNet/EfficientNet）+ 口腔医学特征增强
    ↓
多任务检测头：
    - 病灶检测头（YOLO/FCOS）：定位龋齿、结石等具体位置
    - 分类头：整体健康评分、风险分级
    - 分割头：牙齿分割、牙龈线提取
    ↓
后处理：置信度校准 → 医学逻辑校验 → 可解释性标注
    ↓
输出：结构化报告 + 可视化标注 + 风险等级 + 建议
```

#### 2.1.3 置信度与分级标准

| 风险等级 | 置信度阈值 | 处理方式 | 示例文案 |
|----------|------------|----------|----------|
| **绿色-健康** | >85% 且 无异常 | 常规提醒 | "您的口腔状况良好，建议保持现有护理习惯" |
| **黄色-关注** | 60-85% 或 轻度异常 | 加强护理建议 | "发现轻微问题，建议改善刷牙方式并加强护理" |
| **橙色-预警** | 有明确异常但<90% | 建议就诊 | "建议2周内预约口腔检查，让医生进一步评估" |
| **红色-风险** | >90% 且 明确病灶 | 紧急提醒+导诊 | "发现需要关注的问题，建议尽快预约医生" |

### 2.2 智能拍照引导系统

#### 2.2.1 拍照位标准（5位法）

```
【标准拍摄位】

1. 【正面咬合位】
   - 要求：牙齿自然咬合，露出上前牙和下前牙
   - 用途：牙齿排列、前牙健康、咬合关系
   - 技巧：嘴唇自然放松，可用手指轻拉嘴角

2. 【上牙颌位】
   - 要求：张大口，拍摄完整上牙弓
   - 用途：上牙龋齿、上牙龈健康、智齿
   - 技巧：头稍后仰，手机从下方45度角拍摄

3. 【下牙颌位】
   - 要求：张大口，拍摄完整下牙弓
   - 用途：下牙龋齿、下牙龈健康、舌侧情况
   - 技巧：头稍前倾，手机从上方45度角拍摄

4. 【右侧位】
   - 要求：嘴巴微张，展示右侧磨牙区
   - 用途：右侧后牙、右智齿

5. 【左侧位】
 - 要求：嘴巴微张，展示左侧磨牙区
   - 用途：左侧后牙、左智齿

【儿童版简化】3位法：正面+上牙+下牙
```

#### 2.2.2 实时引导技术

| 功能 | 技术方案 | 用户价值 |
|------|----------|----------|
| **AR引导框** | 基于人脸关键点的口腔区域定位 | 知道镜头该对准哪里 |
| **实时质量检测** | 清晰度、光线、遮挡检测 | 拍不好立即提示重拍 |
| **AI辅助对焦** | 口腔区域自动对焦优化 | 解决手机对焦困难 |
| **语音引导** | 拍摄步骤语音播报 | 解放双手，专注拍摄 |
| **镜像辅助** | 实时预览+示例图叠加对比 | 直观模仿标准姿势 |

#### 拍照体验细节

| 环节 | 体验设计 | 目的 |

| **完成时** | 缩略图预览+确认提交 | 给用户掌控感 |

└─ 成本预估：当前问题如果不处理，未来可能需要的治疗费用

呈现形式：
└─ 时间轴可视化 + "未来您可能遇到..."故事化叙述
```

**差异化价值**：从"诊断工具"升级为"健康决策助手"

├─ 风险提示：监管政策变化风险
└─ 适用：MVP验证期

数据合规：
├─ 医疗数据本地化存储
├─ 数据分类分级管理
├─ 敏感操作留痕审计
└─ 符合《数据安全法》《个人信息保护法》
```

### 5.2 免责声明设计

#### 核心原则
```
1. 不诊断：AI分析仅供参考，不构成诊断

3. 有兜底：明确标注何时必须看医生
4. 可追溯：用户确认已阅读声明
```

#### 关键场景文案

• 本应用建议的"橙色/红色风险"仅为风险提示，需医生进一步确认

我已阅读并理解以上声明

第三方共享限制：
├─ 未经用户明示同意，不向第三方共享个人数据
